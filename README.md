# diTERRA MEDIA — Landing page

Protótipo funcional single page em HTML5, CSS e JavaScript.

## Antes de publicar

1. Adicionar os ficheiros licenciados `CircularStd-Book.woff2` e `CircularStd-Bold.woff2` em `assets/fonts/`.
2. Substituir os fundos temporários por fotografias reais e adicionar `hero-poster.jpg` e `og-cover.jpg`.
3. Adicionar `showreel.webm` e `showreel.mp4` otimizados em `assets/videos/`.
4. Confirmar contactos, logótipos, autorização de uso e textos dos projetos.
5. Ligar o formulário a um backend ou serviço de email.
6. Substituir o URL canónico no HTML, sitemap e robots.
7. Criar a Política de Privacidade e rever proteção de dados.
8. Executar Lighthouse após alojamento e otimização final dos media.

## Teste local

Abra `index.html` diretamente ou execute um servidor local, por exemplo `python3 -m http.server`.
